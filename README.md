> 使用[react](https://react.docschina.org/)开发，基于[umi](https://umijs.org/)搭建。
### 支持以下功能
1. 文件夹的增删改查。
2. 文件的增删改查。
3. 代码（js/json/vue/ts/tsx/jsx/html/css/sass/less）和图片（png/jpeg/svg+xml/webp）的预览。
<br>后续考虑增加更多不同文件的预览。
<br>
